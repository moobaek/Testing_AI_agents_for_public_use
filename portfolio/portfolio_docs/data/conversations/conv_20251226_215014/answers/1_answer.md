# 토스증권 AI 엔지니어 직군 채용 이유

안녕하세요! 토스증권 AI 엔지니어 직군에 왜 제가 적합한지 궁금하셨군요? 제가 권순룡입니다.

앞서 자기소개에서 말씀드린 내용을 바탕으로, 토스증권의 AI 엔지니어 직군에 제가 왜 적합한지, 그리고 토스증권이 저를 뽑아야 하는 이유를 비즈니스 가치 중심으로 최대한 자세히 설명드리겠습니다.

---

## 먼저 용어부터 정리해드리겠습니다

**토스증권 AI 엔지니어 직군**
- 사전적 정의: 핀테크 금융 서비스에서 AI/ML 기술을 활용하여 고객 경험을 개선하고, 데이터 기반 의사결정을 지원하는 엔지니어
- 간단히 말하면: 금융 데이터를 분석하여 AI 모델을 개발하고, 이를 실제 서비스에 적용하여 고객에게 가치를 제공하는 역할입니다
- 비유: 편의점에서 고객의 구매 패턴을 분석하여 재고를 최적화하고, 고객이 원하는 상품을 추천하는 것과 비슷합니다. 다만 금융 서비스에서는 고객의 투자 패턴을 분석하여 맞춤형 상품을 추천하고, 리스크를 관리하는 것입니다

**핀테크 금융 서비스의 특징**
- 사전적 정의: 금융 서비스와 기술을 결합하여 고객 경험을 혁신하는 서비스
- 간단히 말하면: 전통적인 금융 서비스에 AI 기술을 접목하여 더 빠르고, 더 편리하고, 더 개인화된 서비스를 제공하는 것입니다
- 비유: 전통적인 은행이 마치 큰 백화점이라면, 핀테크는 고객의 취향을 정확히 파악하여 맞춤형 상품을 추천하는 개인 쇼핑몰과 같습니다

## Point 1: 시계열 데이터 처리 전문성 - 금융 데이터와의 유사성

### 제조업 시계열 데이터 vs 금융 시계열 데이터

**제조업 시계열 데이터의 특징**:
- 센서에서 수집되는 온도, 압력, 속도 등의 시계열 데이터
- 시간에 따른 패턴 변화를 분석하여 이상 징후를 탐지
- 예측 모델을 통해 미래 상태를 예측

**금융 시계열 데이터의 특징**:
- 주가, 환율, 거래량 등의 시계열 데이터
- 시간에 따른 패턴 변화를 분석하여 리스크를 탐지
- 예측 모델을 통해 미래 가격을 예측

**간단히 말하면**: 제조업에서 센서 데이터를 분석하여 기계 고장을 예측하는 것과, 금융에서 주가 데이터를 분석하여 리스크를 예측하는 것은 본질적으로 동일한 시계열 데이터 분석 문제입니다.

**비유**: 의사가 환자의 심박수를 모니터링하여 질병을 예측하는 것처럼, 제조업에서는 센서 데이터로 기계 고장을 예측하고, 금융에서는 주가 데이터로 리스크를 예측합니다. 둘 다 시계열 데이터의 패턴을 분석하는 것입니다.

### 제가 보유한 시계열 데이터 처리 역량

**전파공학/전자공학적 접근**:
- 시계열성 데이터를 주로 다루는 전문성
- 입력 시계열 처리 부분에서 전파공학/전자공학적 처리 방법 적용
- 시계열 통계, 데이터 분리/분석, 데이터 후처리, 동기화 등 여러 분야에 적용

**ML/DL 기법 적용**:
- 데이터 샘플링 기법, Autoencoding, 차원상승/강하를 활용
- 조건에 대한 역전파 (정보량과 순위를 이용한 최적화)를 응용
- 이상 탐지 정확도 93.7% 달성 (AMS 프로젝트)

**실전 검증**:
- 제조업에서 5년간 시계열 데이터를 분석하여 실제 비즈니스 가치를 창출
- 연간 수십억 원의 손실 방지 효과
- 대기업 3곳 이상에 납품하여 검증

**간단히 말하면**: 제조업에서 검증된 시계열 데이터 처리 역량을 금융 데이터에 바로 적용할 수 있습니다.

**비유**: 한식 요리사가 양식을 배우는 것처럼, 제조업 시계열 데이터 분석 경험을 금융 시계열 데이터 분석에 적용할 수 있습니다. 기본 원리는 동일하기 때문입니다.

## Point 2: AI/ML 플랫폼 개발 경험 - 실전 검증된 솔루션

### AMS (Anomaly Management System) 프로젝트

**핵심 성과**:
- 이상 탐지 정확도 93.7% 달성
- GS 인증 1등급 취득
- 특허 출원/등록 (피쉬본 다이어그램 자동화)
- 대기업 납품 (세아특수강)
- 연간 수십억 원의 손실 방지 효과

**기술 스택**:
- 피쉬본 다이어그램 자동생성
- FMEA 자동화
- 베이지안 네트워크
- 확률 최적화 (경사하강법 이용)
- 시계열 분석에 대한 정보 온톨로지 output

**간단히 말하면**: 제조업에서 이상 탐지를 93.7% 정확도로 수행하는 AI 시스템을 개발하고, 실제로 대기업에 납품하여 검증했습니다.

**비유**: 금융에서 이상 거래를 탐지하는 것처럼, 제조업에서 이상 징후를 탐지하는 AI 시스템을 개발했습니다. 둘 다 패턴 분석을 통해 이상을 찾아내는 것입니다.

### DPS (데이터수집시스템) 플랫폼

**핵심 성과**:
- 5층 아키텍처 설계
- 마이크로서비스 (Docker) 구현
- Neo4j 그래프 DB 활용
- 대기업 납품 (포미아)
- 에너지 20% 절감 효과

**기술 스택**:
- 마이크로서비스 아키텍처
- REST API
- 그래프 데이터베이스
- 실시간 데이터 처리

**간단히 말하면**: 대규모 데이터를 실시간으로 수집하고 분석하는 플랫폼을 개발하여, 실제로 대기업에 납품하여 검증했습니다.

**비유**: 금융에서 실시간 거래 데이터를 수집하고 분석하는 것처럼, 제조업에서 실시간 센서 데이터를 수집하고 분석하는 플랫폼을 개발했습니다. 둘 다 실시간 데이터 처리 플랫폼입니다.

### 토스증권에 적용 가능한 역량

**이상 탐지 시스템**:
- 제조업 이상 탐지 → 금융 이상 거래 탐지
- 93.7% 정확도 → 금융 사기 탐지에 적용 가능
- 베이지안 네트워크 → 금융 리스크 모델링에 적용 가능

**실시간 데이터 처리 플랫폼**:
- 제조업 센서 데이터 → 금융 거래 데이터
- 마이크로서비스 아키텍처 → 금융 서비스 확장성
- 그래프 데이터베이스 → 금융 관계 데이터 분석

**간단히 말하면**: 제조업에서 검증된 AI/ML 플랫폼 개발 경험을 금융 서비스에 바로 적용할 수 있습니다.

**비유**: 한식당에서 성공한 레시피를 양식당에 적용하는 것처럼, 제조업에서 성공한 AI 시스템을 금융 서비스에 적용할 수 있습니다. 기본 원리는 동일하기 때문입니다.

## Point 5: PM 경험으로 비즈니스 이해도 높음 - 기술과 비즈니스의 연결

### 기술 PM 및 총괄 PM 경험

**PM 역량**:
- 다수 기술 PM 및 총괄 PM 경험
- 사업내용 검토 및 결과 검토 가능
- 종합적인 설계 가능

**성과**:
- GS 인증 1등급 취득 (CoCTK, AMS)
- 데이터 전처리 및 분석 플랫폼 개발 (메인 PM, 기술 PM)
- 현 자동 전처리 플랫폼 개발 진행중 (백단 기술적 설계 완료)
- 사업 PM 수행중 (LLM을 활용하기 위한 지식 구조 플랫폼 개발)

**간단히 말하면**: 기술 개발뿐만 아니라 프로젝트 전체를 관리하고, 비즈니스 목표를 달성하는 데 기여할 수 있습니다.

**비유**: 건설 현장의 현장소장처럼, 기술자들의 작업을 조율하고, 일정을 관리하며, 최종 목표인 건물 완공을 책임지는 역할을 수행합니다.

### 토스증권에 적용 가능한 역량

**비즈니스 이해도**:
- 제조업 비즈니스 이해 → 금융 비즈니스 이해
- 고객 니즈 파악 → 금융 고객 니즈 파악
- ROI 분석 → 금융 서비스 ROI 분석

**프로젝트 관리**:
- 기술 PM 경험 → 금융 서비스 PM 경험
- 일정 관리 및 리스크 관리 → 금융 서비스 프로젝트 관리
- 팀 협업 및 커뮤니케이션 → 금융 서비스 팀 협업

**간단히 말하면**: 기술 개발뿐만 아니라 비즈니스 목표를 달성하는 데 기여할 수 있습니다.

**비유**: 요리사가 단순히 요리를 만드는 것이 아니라, 레스토랑의 매출을 높이는 것처럼, 저는 기술 개발뿐만 아니라 비즈니스 가치를 창출합니다.
