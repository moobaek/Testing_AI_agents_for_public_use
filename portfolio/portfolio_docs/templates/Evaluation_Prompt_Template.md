## 포트폴리오 문서 평가 예시

### Phase 1 평가 프롬프트

```
# 포트폴리오 문서 평가 프롬프트 - Phase 1

## 🎭 역할 설정

**당신의 역할**: 업체 표준 문서 생성 가이드 평가 전문가
**평가 대상**: portfolio/portfolio_docs
**평가 관점**: 업체 표준 문서 생성 가이드(AI)로서의 효과성
**평가 일시**: [YYYY-MM-DD HH:MM:SS]

## 📋 평가 기준

### 1.1 프롬프트 명확성 (30점)

**평가 기준**:
- 각 Phase별 프롬프트가 명확한가? (10점)
- 출력 형식이 구체적으로 정의되어 있는가? (10점)
- 예시가 충분한가? (10점)

**체크리스트**:
- [ ] Phase 0-6의 모든 Step에 프롬프트 존재
- [ ] 각 프롬프트에 "출력 형식" 섹션 포함
- [ ] 올바른/잘못된 예시 제공

[나머지 평가 항목...]

## 📊 평가 방법

1. 평가 대상 문서 분석
2. 평가 기준에 따른 점수 부여
3. 개선 제안 작성

## 📝 출력 형식

# Phase 1: 업체 표준 문서 생성 가이드 평가 리포트

**평가 대상**: portfolio/portfolio_docs
**평가 관점**: 업체 표준 문서 생성 가이드(AI)
**평가 일자**: YYYY-MM-DD

## 📊 Executive Summary

### 종합 점수
- **최종 점수**: XX/100
- **등급**: [S/A/B/C/D]

[나머지 리포트 구조...]
```
